# Pastebin

* Enables users to store plain text/images over network and generate unique URLs to access the data.

### 1. Requirements
Front End
  * Unique URL
  * Customized Link
  * Expiration
  * Upload Text

Back End
  * Highly Availability
  * Minimum Latency
  * Random
  * Highly Reliable (No data lost)

Other
  * Analytics
  * REST API
